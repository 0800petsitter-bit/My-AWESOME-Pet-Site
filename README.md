## Getting Started

### Install dependencies

```bash
npm install
```

### Development

Start the development server with hot reload:

```bash
npm run dev
```

### Build

Build for production:

```bash
npm run build
```

### Preview

Preview the production build:

```bash
npm run preview
```

### Lint

Run ESLint to check code quality:

```bash
npm run lint
```
